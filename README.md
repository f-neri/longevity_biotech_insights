# Aging Biotech Insights (ABI)

An interactive dashboard to explore the aging biotech landscape using data from agingbiotech.info

## Getting Started

```bash
git clone https://github.com/f-neri/aging_biotech_insights
cd aging_biotech_insights
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r requirements.txt
python abi_app/app.py
```
